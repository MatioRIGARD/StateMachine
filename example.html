This example structure

This example is following the state machine drawn on the picture above.

States can contain others, like "state1" contains "state2" and "state7".

"h1" and "h2" are history states and are used to recover from an interruption.

You can use a transition like the "bp3" transition to transit from a state to enter another step. In this case, when you enter the history state "h1", you go back to the state you were when you emited the "bp3" signal.
